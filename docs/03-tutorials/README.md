## 💡 使用技巧

### 遇到问题时

1. **先查看教程** - 大部分问题都有解决方案
2. **查看故障排查文档** - 如 `SUPABASE_MCP_SETUP.md`
3. **搜索错误信息** - 在文档中搜索关键词
4. **询问 AI** - 在 Kiro 中直接提问

### 学习建议

- 按顺序阅读教程
- 边学边实践
- 记录遇到的问题和解决方案
- 分享经验给团队
